# edufemme
**Edufemmedafriq** est une plateforme dynamique dédiée à l'**autonomisation des femmes** en Afrique. Elle mise sur l'éducation, le mentorat et l'inclusion numérique pour briser les barrières sociales. En offrant des formations clés en leadership et technologies, elle transforme le potentiel féminin en moteur de croissance.
